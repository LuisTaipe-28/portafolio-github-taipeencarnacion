# Proyecto Web

Página web estática con diseño moderno que incluye navbar, hero, sección de servicios, estadísticas animadas y contacto. No requiere instalación — solo abre `index.html` en tu navegador.

## Tecnologías

- **HTML5** — estructura semántica
- **CSS3** — estilos, animaciones y diseño responsivo
- **JavaScript** — interacciones y efectos de scroll
